# Basic Structure
```html
<!DOCTYPE html>
<html>
  <head>
    <title></title>
  </head>
</html>
```

- **<!DOCTYPE html>** HTML5 문서 선언
- **<html>···</html>** 문서의 시작과 끝
- **<head>···</head>** 메타데이터 부분
- **<body>···</body>** 출력되는 정보 부분

- **<meta charset="UTF-8">** character setting, 한글 깨짐 방지
- **<title>···</title>** 제목
- **<a href="https://···">···</a>** 다른 사이트로 이동
- **<img>** 이미지 삽입. 닫힘 태그 없음
- **<h1>···</h1>** 제목과 부제목들. 숫자가 작을수록 크기가 작아짐. --h1은 한 번만 사용.--
